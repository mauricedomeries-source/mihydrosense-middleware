# 📊 MiHydroSense Evidence

This folder contains verified proof of the MiHydroSense real-time hydroponic monitoring platform, demonstrating full system development, deployment, and operation.

## 🔍 Evidence Included

### 1. Live Dashboard (dashboard-live.png)
- Real-time monitoring interface
- Displays pH, EC, and temperature data
- Hosted and publicly accessible system
- Demonstrates production-level frontend deployment

---

### 2. WeWeb Frontend Builder (weweb-builder.png)
- Visual development environment
- Custom UI components and data binding
- Real-time integration with Supabase backend
- Demonstrates frontend engineering ownership

---

### 3. Supabase Database (supabase-data.png)
- Structured storage of sensor telemetry
- Time-series data with timestamps
- Backend data modelling and persistence layer
- Demonstrates cloud database architecture

---

### 4. MQTT Broker Pipeline (mqtt-broker.png)
- Real-time IoT message ingestion via MQTT (EMQX)
- Device-to-cloud communication pipeline
- Topic-based multi-tenant data streaming
- Demonstrates IoT system architecture
